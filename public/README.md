# Инструкция по добавлению аудиофайлов

Разместите аудиофайлы в папке `/public/audio/` с названиями:
- story1.mp3
- story2.mp3
- story3.mp3
- story4.mp3
- story5.mp3
- story6.mp3
- story7.mp3
- story8.mp3

Поддерживаемые форматы:
- .mp3
- .m4a
- .ogg
- .wav

Для наилучшей совместимости со всеми браузерами рекомендуется использовать формат .mp3
